В языке Python нет специального оператора множественного выбора. Множественный выбор реализуется оператором if, включающим ветви elif (else if) для разбора вариантов.

Синтаксис оператора if Python в случае множественного выбора имеет вид:

if логическое выражение:
	операторы if ветви
elif логическое выражение:
операторы elif ветви
else:
операторы else ветви
Ветвь elif может присутствовать многократно. Ветвь else - ветвь по умолчанию - может отсутствовать.
